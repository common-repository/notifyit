Notifyit is a beautiful notification display plugin for your website.

== Description ==

Notifyit is a simple plugin that enables you to add animated notification to your website.

Major features in Notifyit includes:

* Set frontend notification.
* It has option to set delay time for notification display.
* It has option to choose notification appearance style.

== Frequently Asked Questions ==

= Can I put link in the notification? =

Yes, you can.

== Installation ==

1. Automatic
Go to Dashboard and under Plugins menu click 'Add new' and upload notifyit zip file.

1. Manually
Upload it on wp-content/plugin/ directory
Activate it, and set notifyit options in 'Notify It' menu under Settings menu in Dashboard.

You're done! And you will see notification on the frontend.
